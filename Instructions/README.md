# Build Docs (For Claude Code)

> These files are the build instructions for Claude Code. The user-facing README is at the project root (`../README.md`), generated by Phase 10.

## How to use

### Start a fresh build
```
Read docs/00_OVERALL_PLAN.md and docs/01_PHASE_SCAFFOLDING.md.
Execute Phase 1. Stop after the verification block.
Do not proceed to Phase 2 until I confirm.
```

### After Phase 1 verified
```
Read docs/02_PHASE_DATABASE.md and execute Phase 2. Stop after verification.
```

Continue this pattern through Phase 10. After Phase 10:
```
Read docs/CHECKLIST.md and run every check. Report any failures.
```

## Token budget

Each phase file is ~1,500 tokens. Loading `00_OVERALL_PLAN.md` + a single phase file uses ~3,500 tokens — much less than loading a single 10,000-token monolith every session.

For phases that reference patterns, also load `PATTERNS.md` (adds ~1,000 tokens).
